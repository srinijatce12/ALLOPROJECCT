# Allobaby & Allokonnect Landing Website

## Overview

This project is a responsive React.js landing website developed for the Allobaby and Allokonnect applications.

The website provides detailed information about both applications, including:

* Product Overview
* Key Features
* User Benefits
* Application Workflow
* Screenshots
* Frequently Asked Questions
* Download Information

## Technologies Used

* React.js
* React Router DOM
* Vite
* CSS3
* Vercel

## Project Structure

src/

* components/
* pages/
* App.jsx
* main.jsx

public/

* Images
* Videos
* Icons

## Installation

Clone the repository:

git clone https://github.com/srinijatce12/ALLOPROJECCT.git

Install dependencies:

npm install

Run development server:

npm run dev

Build for production:

npm run build
